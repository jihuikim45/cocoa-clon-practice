## 마우스 모양 바꾸는 방법

☝

```html
#login-form input[type="submit"] { background-color: var(--yellow); cursor:
pointer; }
```

⦸

```html
#login-form input[type="submit"] { background-color: var(--yellow); cursor:
not-allowed; }
```

⟳

```html
#login-form input[type="submit"] { background-color: var(--yellow); cursor:
progress;
```
